# 智能小说爬虫下载器 — 项目说明书

## 一、项目概述

**项目名称**：智能小说爬虫下载器
**项目定位**：快穿/无限流/综影视类型小说专用搜索引擎与自动下载工具
**技术架构**：Flask Web 后端 + 原生 HTML/JS 前端 + 异步爬虫引擎
**运行环境**：Windows / Python 3.10+

---

## 二、功能特性

### 2.1 核心功能

| 功能 | 说明 |
|------|------|
| 多关键词搜索 | 支持同时搜索多个关键词，标签式管理 |
| 13个处理器76个搜索入口 | 覆盖笔趣阁系、帝国CMS系、TXT下载站、通用搜索等 |
| 智能正则过滤 | 多层正则匹配（书名/简介/正文），精准识别快穿类型 |
| 自动下载 | 搜索后自动下载匹配的小说 |
| 手动选择下载 | 仅搜索模式下可手动勾选下载 |
| 去重机制 | 标题去重 + URL去重 + 去重复文件夹对比 |
| 版本替换 | 新版本章节更多时自动替换旧版本 |
| 翻页搜索 | 自动翻页获取更多搜索结果 |
| 实时进度 | 800ms轮询刷新搜索/下载进度 |
| 下载历史 | SQLite持久化存储下载记录 |

### 2.2 搜索模式

- **搜索+下载**：搜索匹配后自动下载，一键完成
- **仅搜索**：只搜索不下载，结果可手动勾选后批量下载
- **单本下载**：搜索结果中可单独下载某一本

### 2.3 站点选择

支持按站点筛选搜索范围，可选择全部站点或指定站点组合。

## 四、模块详解

### 4.1 Flask 后端（main.py）

提供 14 个 API 路由：

| 路由 | 方法 | 功能 |
|------|------|------|
| `/` | GET | 前端页面 |
| `/api/config` | GET/POST | 读取/保存配置 |
| `/api/run` | POST | 启动搜索+下载 |
| `/api/search` | POST | 仅搜索 |
| `/api/stop` | POST | 停止当前任务 |
| `/api/progress` | GET | 实时进度 |
| `/api/results` | GET | 搜索结果（分页+过滤） |
| `/api/download` | POST | 批量下载选中 |
| `/api/download-single` | POST | 单本下载 |
| `/api/download-url` | POST | 获取下载链接 |
| `/api/history` | GET | 下载历史 |
| `/api/stats` | GET | 统计信息 |
| `/api/logs` | GET | 运行日志 |
| `/api/dedup/stats` | GET | 去重统计 |
| `/api/sites` | GET | 站点列表 |

### 4.2 爬虫引擎（engine.py）

**核心流程**：关键词 → 逐站点搜索 → 正则过滤 → 去重检查 → 获取下载链接 → 串行下载

- **串行保证**：每本小说前一本完成后才开始下一本，避免并发问题
- **跨站点去重**：标题规范化后比对（去空格/标点/数字）
- **重试机制**：下载失败自动重试3次，指数退避
- **版本替换**：新版本章节更多时替换去重复文件夹中的旧版本

### 4.3 正则过滤器（filter.py）

三级匹配策略：

| 级别 | 匹配对象 | 规则 |
|------|----------|------|
| 书名关键词 | 标题 | 快穿/无限流/综影视/综漫/穿书攻略等 |
| 简介标签 | 简介/标签 | 【快穿】/快穿+/炮灰攻略系统/世界任务等 |
| 正文扫描 | 正文内容 | 强信号（多世界切换）单独命中 / 弱信号+辅助词双重验证 |

### 4.4 下载管理器（downloader.py）

- 异步流式下载，支持断点续传（.part临时文件）
- 文件大小限制（默认50MB）
- 内容验证：下载后抽样检查是否匹配正则
- 自动重试（3次，指数退避）

### 4.5 去重管理器（dedup.py）

- 扫描"去重复文件夹"建立书名索引
- 支持多种文件名格式解析（编号型/日期型/纯书名）
- 模糊匹配：规范化标题后检查包含关系（60%重叠率）
- 章节数对比：新版本章节更多→替换，否则→跳过

## 六、配置说明

### 6.1 config.json 结构

```json
{
  "search": {
    "keywords": ["快穿", "无限流", ...],
    "regex_filter": { ... },
    "max_pages": 5,
    "results_per_site": 999999
  },
  "crawler": {
    "concurrency": 3,
    "request_delay": 1.5,
    "timeout": 30,
    "retry_count": 3
  },
  "download": {
    "output_dir": "downloads",
    "max_file_size_mb": 50,
    "verify_content": true,
    "verify_sample_bytes": 50000,
    "organize_by": "category"
  },
  "server": {
    "host": "127.0.0.1",
    "port": 8765,
    "auto_open": true
  }
```

### 6.2 配置项说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| search.keywords | ["快穿",...] | 搜索关键词列表 |
| search.max_pages | 5 | 每个站点最大翻页数 |
| search.results_per_site | 999999 | 每站最大结果数 |
| crawler.concurrency | 3 | 最大并发数（实际串行=1） |
| crawler.request_delay | 1.5 | 请求间隔（秒） |
| crawler.timeout | 30 | 请求超时（秒） |
| crawler.retry_count | 3 | 下载重试次数 |
| download.output_dir | downloads | 下载输出目录 |
| download.max_file_size_mb | 50 | 最大文件大小 |
| download.organize_by | category | 归档方式（category/author/site） |
| server.host | 127.0.0.1 | 服务器地址 |
| server.port | 8765 | 服务器端口 |
| server.auto_open | true | 启动时自动打开浏览器 |

## 七、使用指南

### 7.1 安装

```bash
cd novel_crawler
pip install -r requirements.txt
```

依赖列表：
- flask
- flask-cors
- aiohttp
- beautifulsoup4
- aiofiles
- lxml

### 7.2 启动

```bash
python main.py
```

启动后自动打开浏览器访问 `http://127.0.0.1:8765`

### 7.3 使用流程

1. **设置关键词**：使用默认标签或输入自定义关键词
2. **选择站点**：点击站点Chip选择/取消，默认全部站点
3. **开始搜索**：
   - 点击"开始搜索"→ 搜索+自动下载
   - 点击"仅搜索"→ 只搜索，结果可手动选择下载
4. **查看结果**：搜索结果标签页实时更新
5. **手动下载**：勾选需要的小说，点击"下载选中"
6. **查看日志**：切换到"运行日志"标签页查看详细日志
7. **查看历史**：切换到"下载历史"标签页查看历史记录
8. **修改配置**：切换到"配置"标签页调整参数

### 7.4 去重复文件夹

在 `downloads/去重复文件夹/` 目录中放入已有的小说文件，系统会：
- 自动扫描建立索引
- 搜索到同名小说时对比章节数
- 新版本章节更多→替换旧版本
- 新版本不更多→跳过下载

- **search_cache**：搜索缓存（关键词/站点/结果JSON）
- **crawl_state**：爬取状态（任务ID/状态JSON）

## 九、注意事项

1. **网络环境**：部分站点可能需要代理，在config.json的proxy段配置
2. **请求频率**：默认1.5秒间隔，过快可能被封IP
3. **站点可用性**：部分站点可能临时不可用，属正常现象
4. **文件编码**：下载文件自动处理编码，部分乱码为源站问题
5. **去重文件夹**：首次使用前需手动放入已有小说文件
6. **端口冲突**：如8765端口被占用，修改config.json中的port值

## 十、API 接口文档

### 10.1 启动搜索+下载

```
POST /api/run
Body: { "keywords": ["快穿"], "site_names": null, "auto_download": true }
Response: { "ok": true, "message": "串行爬虫已启动" }
```

### 10.2 仅搜索

```
POST /api/search
Body: { "keywords": ["快穿"], "site_names": null }
Response: { "ok": true, "message": "搜索已启动" }
```

### 10.3 获取搜索结果

```
GET /api/results?per_page=20&page=1&status=&source=
Response: {
  "results": [...],
  "pagination": { "page": 1, "per_page": 20, "total": 100, "total_pages": 5 },
  "sources": ["笔趣阁", "知轩藏书", ...],
```

```
Body: { "results": [{ "title": "...", "url": "...", "source": "..." }] }
```
